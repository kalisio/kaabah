    # What is it ?

    Kaabah is a Terraform based solution to build and operate Docker Swarm infrastructures either on Scaleway or on AWS.

    In order to help you operate your cluster easily, these infrastructures are bootstrapped with the following services:

    • Registry: a private registry to use as part of your infrastructure
    • Traefik: the native cloud edge router !
    • Portainer: a simple management UI for Docker
    • Prometheus: a monitoring system
    • Grafana: the analytics platform for all your metrics
